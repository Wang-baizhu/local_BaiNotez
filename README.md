### 使用
- 确保已安装Anaconda
- 执行run.bat
- 在APP填入

    - 本地LLMURL：http://任何支持openai标准接口的api＋端口（如ollama：http://本地IP:11434/v1）
    - 模型名称
    - API key：(ollama)
